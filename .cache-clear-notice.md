# 浏览器缓存清理说明

如果遇到 "Button is not defined" 错误，这是浏览器缓存问题。

## 解决方法：

1. **硬刷新浏览器**：
   - Windows/Linux: Ctrl + Shift + R
   - Mac: Cmd + Shift + R

2. **清除浏览器缓存**：
   - 打开开发者工具 (F12)
   - 右键点击刷新按钮
   - 选择"清空缓存并硬性重新加载"

3. **重启开发服务器**：
   ```bash
   pnpm run dev:h5
   ```
