# TaskPro 微信小程序需求文档

## 1. 应用概述

### 1.1 应用名称

TaskPro

### 1.2 应用描述

TaskPro 是一个轻量级任务管理系统，提供类似聊天的任务输入体验和强大的标签管理功能，帮助用户高效管理任务和项目。\n

## 2. 核心功能

### 2.1 话题管理

- 创建和管理不同的话题（项目）\n- 支持全局搜索话题
- 每个话题左侧显示话题图标，右侧显示话题名称和话题简介
- 可以新增、修改、删除话题
- 删除话题后，该话题对应的任务也会被删除

### 2.2 任务管理

#### 2.2.1 Chat-style 输入

- 像聊天一样快速创建任务
- 支持自然语言输入任务内容

#### 2.2.2 智能标签

- 支持 `#标签/子标签` 格式自动解析
- 可为任务添加多个标签
- 支持标签分类和层级结构
- 支持标签的新增、修改和删除

#### 2.2.3 任务状态

- 支持任务完成标记
- 支持置顶任务
- 支持收藏任务
- 支持任务编辑、删除

#### 2.2.4 附件支持

- 支持图片上传与预览
- 支持附件上传
- 图片等附件存储到 S3 中
- 用户登录后可设置自己的 S3 配置

## 3. 用户登录

- 支持微信授权登录
- 支持账号密码登录，账号必须为邮箱、密码必需使用数字和字母，不小于6位数，密码中可以保护特殊字符，不强制
